If JOGL Environment variables are not set, the batch file "JOGL.bat" can be used for temporarily setting the classpath from the cmd

The main class is SpaceshipInvasion.java
Assuming JOGL is correctly installed, it can be re-compiled with the following command:
"javac SpaceshipInvasion.java"

The program can be run through cmd using the command
"java SpaceshipInvasion"


The key "F" may be used to toggle the game's current frame rate during gameplay.

A video demonstration of the game can be seen at: (https://www.youtube.com/watch?v=_qUsChd77Dk)
